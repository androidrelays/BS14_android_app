# BS14 Launch Screen Icon - README

## About This Icon
This is a custom-designed launch screen icon specifically created for the BS14 app. It features:

- **Large "BS14" text** for maximum readability
- **"By Relport, Inc." branding** for professional appearance
- **Blue gradient background** matching the app theme
- **High contrast white text** for clarity
- **Decorative lines** for visual appeal

## How to Use
1. Open `launch_screen_branded.html` in any web browser
2. Click "Generate Branded Launch Icon" 
3. Click "Download Launch Icon" to save the PNG file
4. Save the downloaded file as `launch_screen_icon.png` in this directory
5. The app configuration will automatically use this icon for the splash screen

## Design Features
- **512x512 pixels** for high resolution
- **Professional typography** with clear hierarchy
- **Shadow effects** for depth and readability
- **Corporate branding** with Relport, Inc. attribution
- **Blue theme consistency** with app colors
